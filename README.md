# XpanderFX
JavaFX 2048 Game.

## How to play?
The UP(or W), DOWN(or S), RIGHT(or D), and LEFT(or A) buttons(keyboard) to control tiles in different directions.

The scene's arrow buttons can't be use(i'm coding on it) to control tiles.

## Requirements
Java 8
